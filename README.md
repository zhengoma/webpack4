# webpack4demo
one by one step to make a system

#下面开始是命令行

- 全局安装 
> npm install -g webpack webpack-cli

- 初始化
> npm init -y

- 创建三个文件夹
mkdir src

- 创建文件
> touch index.html
> touch src/index.js

*webpack4.x中打包默认找src/index.js作为默认入口，可以直接在终端中输入命令webpack 将当前的内容进行一个简单的打包*
